# ZiMin Ottomad Edit
> 指针空转一小时，打你就像打蚂蚁

基于原模组的重构增强版本，并将素材全部替换为籽岷。 <br/>
由于原始模组的随机函数写的不太好，且原作者弃坑，于是开了个Fork重构了。 <br/>

## 食用指北
为了加载本模组，以下内容是必须的：
- Minecraft 1.21.1
- Fabric Loader
- Fabric API
- Java 17+ (建议Java 21)

为了达到最佳配置和使用效果，以下模组可选：
- Mod Menu
- Cloth Config
